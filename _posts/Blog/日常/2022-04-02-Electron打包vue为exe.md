2. Node.JS下载安装
    默认一路向下
    https://nodejs.org/zh-cn/download/

3. 打开cmd命令安装vue(分别输入输入如下命令)
   
   ```
   npm install vue
   npm install -g @vue/cli
   ```

4. cmd定位到需要打包的文件夹下
   
   ```
   npm install（如果不行就用npm install -force）
   npm start(开始)
   npm run build
   ```
